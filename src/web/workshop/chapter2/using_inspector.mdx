Job Inspector is a tool in Splunk Web that you can use to troubleshoot search performance and understand the behavior of knowledge objects such as event types, tags, lookups, and so on within the search. You can inspect search jobs that are currently running or that have finished recently.


Job inspector can be invoked by clicking on "Job -> Inspect Job" under the search bar. 
Or you can use your search id in the following URL: .../en-GB/manager/splunk4champions2/job_inspector?sid=1673511581.496
