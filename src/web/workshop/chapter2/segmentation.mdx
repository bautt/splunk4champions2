## What is event segmentation?

What happpens when data arrives in the index? Among other thing it gets segmented in tokens and tokens will be stored in TSIDX files. Major breakers define tokens. 
Tokens with minor breakers will be stored as a TERM and in separate tokens. 
So the first step in segmentation process is to break up _raw using major breakers. Next the minor breakers are applied to the tokens created by major breakers. Both are stored in the TSIDX file.
