A reference guide for Search Processing Language commands and syntax, accessible directly from the search bar. When active, Search Assistant displays typeahead suggestions as you type — command syntax, usage examples, a history of recent searches using that command, and a link to documentation.

**Search Assistant has three modes:**

| Mode | Behaviour |
|---|---|
| **Full** | Shows command syntax, usage examples, and recent search history |
| **Compact** | Shows typeahead only — less screen space used |
| **None** | Disables Search Assistant entirely |

**How to change:** Settings (gear icon in search bar) → Search Preferences → **Search assistant**
