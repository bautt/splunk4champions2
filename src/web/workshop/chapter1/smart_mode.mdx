**Smart mode** adapts based on your search: if there are no transforming commands, it behaves like Verbose and extracts all fields. If transforming commands (`stats`, `chart`, `timechart`) are present, it behaves like Fast and skips raw event retrieval entirely. This gives you the best balance of speed and information. **Tip: Smart mode is the recommended default.**
