## How to Set Up Splunk Mobile

### 1. Download Splunk Mobile
Download the app suitable for your device:

- [iOS (iPhone)](https://apps.apple.com/us/app/splunk-mobile/id1420299852)
- [Android](https://play.google.com/store/apps/details?id=com.splunk.android.alerts&hl=en_US&gl=US&pli=1)
- [iPad](https://apps.apple.com/us/app/splunk-for-ipad/id1568523145)

### 2. Set Up Splunk Secure Gateway (SSG)
Included in Splunk Enterprise 8.1+ and Splunk Cloud Platform 8.1.2103+.

- Get started: <Link to="https://docs.splunk.com/Documentation/SecureGateway/3.4.251/Admin/Requirements" openInNewContext>"Splunk Secure Gateway Requirements"</Link>
- Security details: <Link to="https://docs.splunk.com/Documentation/SecureGateway/3.4.251/Admin/Security" openInNewContext>"Secure Gateway Security"</Link>
- Configure SAML auth: <Link to="https://docs.splunk.com/Documentation/SecureGateway/3.4.251/Admin/SAMLauth" openInNewContext>"SAML Authentication"</Link>

### 3. Log Your Device into Splunk
Use the instructions in the **Connected Experiences** app or the manual: <Link to="https://docs.splunk.com/Documentation/SecureGateway/3.4.251/User/RegisterorUnregisteraDevice" openInNewContext>"Register or Unregister a Device"</Link>

### 4. Use Splunk Mobile
Once logged in, you can:

- Explore dashboards and alerts: <Link to="https://docs.splunk.com/Documentation/Alerts/2.36.0/Alerts/About" openInNewContext>"About Splunk Alerts"</Link>
- Send alerts and dashboards to mobile devices: <Link to="https://docs.splunk.com/Documentation/Alerts/2.36.0/Alerts/SendAlerts" openInNewContext>"Send Alerts"</Link>
