## Dashboard Studio: The Next Generation of Splunk Dashboards

Dashboard Studio offers a modern, flexible canvas for designing interactive dashboards with pixel-perfect control, built-in data-driven layouts, and advanced customization.

---

### Key Components

#### Visualizations  
- **ID**: Unique identifier (e.g. `viz_f49snx`)  
- **Type**: Chart type (e.g. Column, Line, Gauge)  
- **Options**: Presentation settings (stack mode, color palettes)  
- **Data Source**: Referenced by data source ID  

#### Data Sources  
- **ID**: Unique identifier (e.g. `ds_j2kn9c`)  
- **Type**: Search type (Inline, Saved Search, Chain Search)  
- **Options**: Search parameters (time range, refresh interval)  

#### Inputs  
- **ID**: Unique identifier (e.g. `input_host`)  
- **Type**: Input control (Dropdown, Textbox, Time Picker)  
- **Options**: Token binding and default values  

#### Layout  
- **Type**: Grid or Absolute positioning  
- **Structure**: Coordinates (x, y) and size (width, height) in pixels  
- **Global Inputs**: Input display order and grouping  

#### Defaults  
- **Visualizations**: Default options (show last update time)  
- **Data Sources**: Default time range and refresh rate

## Data Sources in Studio

- **Inline Searches**: Ad-hoc SPL queries embedded directly in the dashboard.  
- **Saved Searches**: Reuse existing scheduled searches; ensure proper sharing permissions.  
- **Chain Searches**: Define dependent searches where one search feeds results into another, all configured via the UI.

## What’s New in Dashboard Studio

### Splunk Enterprise 9.x  
- **Advanced Theming**: Custom color palettes and responsive layouts  
- **Chained Searches UI**: Drag-and-drop configuration for multi-stage queries  
- **Enhanced Input Controls**: New slider and multi-select dropdown components  
- **Dynamic Drilldowns**: Contextual navigation between panels without manual token wiring  

### Splunk Enterprise 10.x  
- **Vector Shapes & Annotations**: Add custom SVG shapes, arrows, and text overlays directly on visualizations  
- **Conditional Formatting**: Apply rules to change visual appearance based on data thresholds  
- **Custom Layout Breakpoints**: Define responsive behavior for different display sizes  
- **Performance Insights**: Built-in metrics for panel render times and query performance profiling
