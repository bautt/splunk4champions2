## Pixel-perfect Layout 


### Visualizations
* Unique ID -- e.g. viz_f49snx or viz_kpi1
* Type -- e.g. column chart
* Options -- e.g. stack mode, colors
* Data Source -- referenced by ID
### Data sources
* Unique ID -- e.g. ds_j2kn9c or ds_kpi2
* Type -- e.g. saved search
* Options -- e.g. refresh rate 
### Inputs
* Unique ID -- e.g. input_poxm20 or input_host
* Type -- e.g. dropdown
* Options -- e.g. token name 
### Layout
* Type -- e.g. Grid or Absolute
* Structure -- x,y viz location and w,h size (px)
* Global Inputs -- display order of inputs 
### Defaults
* Visualizations - e.g. specific options, show last updated
* Data Sources - e.g. time range, refresh rate
