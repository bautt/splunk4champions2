## Searching Metrics

Metrics have their own set of search commands. Let's explore them using the TA-open-meteo weather data in `s4c_meteo_metrics`.

## Using mcatalog

The `mcatalog` command lets you discover what metric names and dimensions are available in a metrics index. Think of it as the metrics equivalent of `| metadata`.

Discover all metric names in our weather metrics index:
<SplunkSearch earliest="-30d" latest="now" spl={`| mcatalog values(metric_name) WHERE index=s4c_meteo_metrics`}></SplunkSearch>

See which cities are reporting weather data:
<SplunkSearch earliest="-30d" latest="now" spl={`| mcatalog values(city) WHERE index=s4c_meteo_metrics`}></SplunkSearch>

Combine metric names with dimensions to understand the data shape:
<SplunkSearch earliest="-30d" latest="now" spl={`| mcatalog values(metric_name) values(city) values(country) WHERE index=s4c_meteo_metrics`}></SplunkSearch>

## Using mstats

The `mstats` command is the primary way to query metric data. It performs statistical aggregations directly on the metrics store, which is significantly faster than running `stats` on event data.

Average temperature by city over the last 24 hours:
<SplunkSearch earliest="-24h" latest="now" spl={`| mstats avg(meteo.temperature_2m) as avg_temp WHERE index=s4c_meteo_metrics BY city span=1h`}></SplunkSearch>

Wind speed trends with timechart:
<SplunkSearch earliest="-7d" latest="now" spl={`| mstats avg(meteo.wind_speed_10m) as avg_wind prestats=true WHERE index=s4c_meteo_metrics BY city span=4h
| timechart avg(meteo.wind_speed_10m) as avg_wind BY city span=4h`}>
</SplunkSearch>

Humidity and temperature together:
<SplunkSearch earliest="-24h" latest="now" spl={`| mstats avg(meteo.temperature_2m) as avg_temp avg(meteo.relative_humidity_2m) as avg_humidity WHERE index=s4c_meteo_metrics BY city`}></SplunkSearch>

## Using mpreview

Use `mpreview` to get a raw look at the metric time series stored in a metrics index. This is useful for troubleshooting and understanding the data structure.
<SplunkSearch earliest="-1h" latest="now" spl={`| mpreview index=s4c_meteo_metrics`}></SplunkSearch>
