## Metrics

Splunk 7+ supports a dedicated **metric index** optimized for high-volume time-series data.

A **metric** is a timestamped measurement enriched with one or more **dimensions**. Each data point can include multiple measurements (multi-value metrics) at a single timestamp[web:108].

### Time  
Default precision is seconds; millisecond timestamps are configurable per metric index.

### Metric Name  
Dot-separated segments define a metric’s namespace[web:118].

### Numeric Values  
Integer or floating-point numbers representing each measurement (e.g., counts, gauges).

### Measurements  
Pairs of metric names and numeric values recorded at a given timestamp; supports multiple measurements per event (multi-value metrics)[web:108].

### Dimensions  
Key–value pairs that add contextual metadata to measurements (e.g., host, region, service).
